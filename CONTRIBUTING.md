# Contributing to HomeschoolIQ

Contributions are welcome at any time. You do not need to know how
to code to contribute to this project.

---

## Suggest a Research Source

The most valuable contribution you can make is a credible source
this project has not included yet.

**To submit a source:**

1. Open a [GitHub Issue](https://github.com/dustindoesdata/homeschool-iq/issues/new)
2. Title it: `Source suggestion: [Publisher or Study Name]`
3. Include the URL and one sentence on what it covers

The maintainer will review it, assign a credibility grade, and
add it to the pipeline if it meets the quality bar.

**What qualifies:**
- Government agency publications (federal or state)
- Peer-reviewed academic studies
- Research organizations with documented methodology
- Established education journalism outlets

**What does not qualify:**
- Personal blogs or opinion pieces
- Undated or unattributed content
- Sources without a named publisher

---

## Report a Problem

If a stat looks wrong, a source link is broken, or a finding
seems misleading — open a
[GitHub Issue](https://github.com/dustindoesdata/homeschool-iq/issues/new)
and describe what you found. Every flag gets reviewed.

---

## Code Contributions

If you want to contribute code, open an Issue first describing
what you want to build. All Python code must pass `flake8`
before a pull request will be merged.

---

Thank you for helping make this resource more useful for
homeschooling families.
